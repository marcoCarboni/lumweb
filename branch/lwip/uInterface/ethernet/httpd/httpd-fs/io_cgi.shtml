I/O Control Demo

UTC

This demonstration shows a method of performing control and status reporting. We use Server Side Include tags to replace text in the page as it is being server from the Stellaris board and standard HTML forms to send data to a CGI handler running on the board.  This example does cause the page to be reloaded whenever form data is submitted but it  involves less complex HTML to perform the task.
